Tamajarly Download Instructions

BACK UP YOUR SAVE FIRST OR YOU MAY LOSE SAVE DATA

Save data is located by typing %APPDATA% in your search bar

1. Download the zip file.

2. Locate the folder in which your current Tamajarly folder is located in.

3. Drag the zip folder into the folder containing the Tamajarly folder (do not put inside the actual Tamajarly folder).

4. Delete the old Tamajarly folder.

5. Unzip the Tamajarly zip file.

6. Delete the Tamajarly zip file.

Download File

Backing up your data

1. Type %APPDATA% into your computer search bar (usually bottom left).

2. Go back a File Directory by clicking AppData in your navigation bar near the top of the window.

3. Go into the LocalLow folder.

4. Go into the KyloCorp folder.

5. Copy the data.game file and send it to yourself (email, discord, etc).
